Websites Used

Mozilla Developer Network

Mozilla's developer oriented website. This site contains documentation for developers looking to make their work compatiable with Firefox.

W3Schools

A large educational website oriented towards learning how to build webpages. Numerous topics on HTML and CSS are covered, and there are extensive resources for other languages like Javascript and SQL, among others.

CSS Tricks

A website devoted entirely to CSS. Tutorials, definitions, ideas, and videos are focused on many CSS topics. Be sure to look at posts about implementing interesting features or troubleshooting.

Wikipedia

Famous encyclopedia website. Useful for understanding various programming concepts and design philosophies. Does not really contain information directly useful for coding but for learning about certain concepts.

Udacity

A large educational website oriented towards learning how to build webpages. Numerous topics on HTML and CSS are covered, and there are extensive resources for other languages like Javascript and SQL, among others.

Specific Websites

HTML Reference

This is a List of all HTML tags, with links to pages detailing how the tags are used with examples.

Mozilla Developer Network: HTML

A detailed landing page for new developers looking to work with HTML in the Firefox browser. Contains links to tutorials for beginners and HTML references.

Tree Structure

Wikipedia entry describing tree structures as a design method. A neccessity for designing HTML pages.

Udacity Style Guide: Type Selectors

This is a guide for styling ones code, with tips for many aspects of HTML. In this case this entry focuses on calls to classes in the CSS file.

How CSS Selectors Work

A guide for understanding and using CSS Selectors, which are what allows you to apply style to specific HTML elements.

CSS Reference

Mozilla's CSS documentation. Covers CSS concepts, selectors, properties, data-types, and documentation useful for understanding CSS. Also gives the default stylesheet for HTML displayed in Firefox.

Webkit Default Stylesheet

The default stylesheet for webkit-based browsers.

Internet Explorer Default Stylesheet

The default stylesheet for Internet Explorer.

Font

Documentation page for the CSS property Font in Firefox. Includes examples and explanations of settings.

HTML Element Reference

Summary of each of HTML's elements and links to detailed entries for all. Includes depreciated elements.

Using HTML Sections and Outlines

Information and advice on how to structure ones HTML for readability and so that outlining algorithms can read your code easier.

Markup Validation Service

Scans HTML code for errors such as missing child elements.

CSS Validation Service

Scans CSS code for errors and suggests possible fixes.
